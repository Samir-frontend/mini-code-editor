## 💻 Overview

> A **mini code editor** built right in the browser — inspired by VS Code's dark interface. Write HTML, CSS & JavaScript and see the **live output instantly**, just like CodePen.

---

## ⚡ Core Features

```js
const features = {
  liveOutput:     "✅ Real-time preview as you type",
  multiPane:      "✅ Separate HTML / CSS / JS editors",
  darkTheme:      "✅ VS Code inspired dark UI",
  syntaxStyling:  "✅ Code-friendly monospace fonts"
};
```
